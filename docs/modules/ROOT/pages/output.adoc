== Configuration

[cols="1,2,1"]
|===
|Option |Description |Example

|`--report-type`
|Output format selection (`text`, `json`, `yaml`, `junit`, `sarif`)
|`--report-type=json`

|`--report-file-name`
|File to write output (defaults to stdout)
|`--report-file-name=report.json`

|`--color`
|Enable/disable a colored output (text format only)
|`--color=false`
|===
